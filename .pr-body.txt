- **TW routing fix**: dividends route now uses `get_fundamentals_provider()` registry so TW symbols route to FinMind instead of yfinance
- **TW cash vs stock split**: yearly totals now include separate cash/stock columns; events distinguish type
- **Frontend DividendCard**: added scrollable "Recent dividends" events table + color-coded payout health badge (blue/green/orange)
