## Summary
Adds FinMindProvider for TW/*.TW/*.TWO symbols behind a symbol-based registry, with graceful degradation when FinMind is unavailable.

- **FinMindProvider**: `income_statement`, `balance_sheet`, `cash_flow`, `dividends` with long-to-wide pivot for financial statements
- **get_fundamentals_provider(symbol)**: routes Taiwan symbols (4-digit or .TW/.TWO) to FinMind; all others use yfinance
- **Lazy initialization**: FinMind package loaded only when `FINMIND_TOKEN` is set; no crash if package absent
- **24-hour cache TTL** for all fundamentals calls (rate-limit friendly ~300-600 req/hr on free tier)
- Added `FINMIND_TOKEN` to `Settings`; `FinMind>=4.0` to `requirements.txt`
- **pandas_ta_compat**: shim so `stocks.py` works when `pandas-ta` is unavailable (Python < 3.12)
